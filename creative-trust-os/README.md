# Creative Trust OS — システム設計図サイト

「AIと信頼でつながる、フリーランスデザイナーチームの制作インフラ」のシステム設計図
（全体アーキテクチャ）を1ページにまとめた静的サイトです。

公開先: https://nmp-creative-trust-os.netlify.app/

## 構成

- `index.html` — 単一ファイルで完結（外部CDN・フォント・画像への依存なし）
  1. ヒーロー + 目指す世界（3つの柱）
  2. 制作フロー全体像（6つのステップ）
  3. システム全体アーキテクチャ図（ユーザー層 / AIエンジン層 / 業務アプリ層 / データ層 / インフラ・外部連携層）
  4. 主要データモデル（ER図・8エンティティ + リレーション）
  5. 詳細データフロー（案件開始からナレッジ蓄積まで）
  6. 主要画面イメージ（5画面）
  7. 外部ツール連携 + セキュリティ設計
  8. この仕組みで実現する価値（クライアント / デザイナー / チーム）

## デプロイ方法

**A. リポジトリ連携（推奨)**: Netlifyでこのリポジトリを接続すると、
ルートの `netlify.toml` により `creative-trust-os/` がそのまま公開されます。

**B. 手動デプロイ**: このフォルダを Netlify Drop にドラッグ&ドロップするだけです。

ビルド工程はありません。`index.html` をブラウザで直接開いても確認できます。
