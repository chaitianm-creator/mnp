# ユーザーフロー

## 1. 初回利用フロー

```mermaid
flowchart TD
    A[LP] --> B[新規登録<br/>メール+パスワード+規約同意]
    B --> C{メール確認<br/>※ローカルは自動確認}
    C --> D[初回オンボーディング]
    D --> D1[1. サービス説明]
    D1 --> D2[2-4. カメラ/ぼかし/マイク不使用の説明]
    D2 --> D3[5. ブラウザ権限リクエスト]
    D3 -->|拒否| D3e[原因とブラウザ別設定方法を表示<br/>カメラなしでも続行可]
    D3 --> D4[6. カメラ位置確認<br/>ぼかしプレビュー]
    D3e --> D4
    D4 --> D5[7. 表示名設定]
    D5 --> D6[8. 今日の学習内容]
    D6 --> D7[9. 5分間の体験セッション]
    D7 --> E[ホーム]
```

## 2. 自習セッションフロー(中心フロー)

```mermaid
flowchart TD
    H[ホーム] -->|今すぐ入室| P[入室前設定<br/>学習内容/時間/カメラ/ルール同意]
    P -->|入室| J{join_room RPC<br/>サーバー側で部屋割当}
    J -->|空き部屋あり| R[既存部屋へ]
    J -->|満室/なし| RN[新規部屋を作成]
    J -->|無料枠超過| PLAN[プラン案内]
    J -->|利用停止中| BAN[入室拒否]
    R --> W[自習室]
    RN --> W
    W --> W1[開始前カウントダウン<br/>案内役の開始メッセージ]
    W1 --> W2[集中タイム<br/>同期タイマー/ぼかし映像共有]
    W2 -->|完了| F[セッション終了画面<br/>自己評価+メモ → 完了コマ記録+XP]
    W2 -->|途中退出<br/>確認ダイアログ| L[実参加時間のみ記録<br/>完了コマにはならない]
    F -->|もう1コマ| P
    F -->|ホームへ/休憩| H
    L --> H
    W2 -->|リロード/切断| W3{セッション有効?}
    W3 -->|残り時間あり| W2
    W3 -->|終了済み| F
```

## 3. 予約フロー

```mermaid
flowchart TD
    H[ホーム] --> RV[予約一覧]
    RV --> RN[予約作成<br/>日付/開始/時間/内容/繰り返し]
    RN -->|無料プラン3件超| UP[アップグレード案内]
    RN --> RS[予約完了<br/>ブラウザ通知許可]
    RS -->|開始前| NT[通知+ホームに強調表示]
    NT -->|直接入室| P[入室前設定へ]
    RS -->|時刻経過・未入室| AB[欠席として記録]
```

## 4. 応援者フロー

```
利用者: メールアドレス登録 → 招待メール送信
応援者: メール内リンク → 同意ページ → 同意
以後: 学習開始/完了時に通知、週間レポート (本人がいつでも停止/削除可能)
```

## 5. 通報・管理フロー

```
利用者: 自習室内で参加者を通報(種別+詳細) → reports に記録
管理者: /admin/reports で確認 → 利用停止/解除 → admin_audit_logs に操作記録
利用停止ユーザー: ログイン時・入室時に拒否
```
