# MVP / 正式版 実装範囲

## MVP (Phase 2) — 本リポジトリで実装

- メール認証(登録/ログイン/再設定/退会)・規約同意
- オンボーディング(体験セッション含む)
- ホーム / 入室前設定 / カメラテスト
- 自習室: フルメッシュWebRTC(最大6名)・強制ぼかし・マイク無効・同期タイマー・再接続・多重入室防止
- セッション記録(完了コマ/実参加時間)・セッション終了画面
- 学習履歴ダッシュボード(日/週/月・カレンダー・傾向・達成率)
- 予約(単発+毎週繰り返し)・ブラウザ通知・欠席記録・無料3件制限
- 習慣化(XP/レベル/連続日数/バッジ/週間・月間目標) — サーバー側付与
- 応援者(メール招待・同意・通知・停止) — メール送信はResendキー設定時のみ実送信
- 通報・ブロック
- 管理者画面(ユーザー/ルーム/通報/お知らせ/集計/監査ログ)
- プラン管理とStripe Checkout/Portal接続(テストモード・キー設定時に有効)
- レスポンシブ・ダークモード
