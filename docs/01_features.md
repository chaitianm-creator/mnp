# 機能一覧(調査結果と本サービスの機能定義)

## 1. 参考サービスの公開情報調査

参考サービス(オンライン自習室)の公開ページは、本開発環境からの直接アクセスが
HTTP 403 で拒否されたため、検索エンジン経由で確認できた公開情報(公式紹介ページの
検索結果要約、第三者のレビュー記事・報道記事の要約)を基に整理した。

### 確認できた機能(公開情報ベース)

| # | 機能 | 出典の種類 |
|---|------|-----------|
| 1 | 入室すると自動でぼかし付きカメラがON、マイクは自動でOFF | 公式紹介/レビュー |
| 2 | 映像には個人が特定されない強さのぼかしが入る | レビュー/報道 |
| 3 | 25分を1コマとする学習時間 | レビュー |
| 4 | 最大6名程度が同じ画面に表示され、同じ時間を共有して自習する | レビュー |
| 5 | 事前予約制(4:00〜26:00の枠から選択) | レビュー |
| 6 | 定期的に決まった時間の繰り返し予約が可能 | レビュー |
| 7 | 予約と参加の履歴が確認できる | レビュー |
| 8 | 月額課金(公開情報では金額に幅あり。980円/月という記載と3,000円/月という記載が混在) | レビュー |
| 9 | 「やる気に頼らない」ことをコンセプトとした習慣化設計 | 公式/報道 |
| 10 | 資格学校との提携プランの存在 | 公式提携ページ |

### 要確認(会員専用のため公開情報から確定できない機能)

以下は推測で断定せず「要確認」とする。本サービスでは要件定義書の指示に従い独自に設計した。

- 入室後の画面レイアウトの詳細、参加者情報の表示項目
- タイマーの同期方式、途中退出時の扱い
- 通知(メール/LINE/プッシュ)の具体的な種類とタイミング
- 応援・見守り機能の有無と仕様
- レベル・バッジ等のゲーミフィケーションの有無と仕様
- 通報・ブロック等の安全機能の詳細
- 無料枠の正確な範囲、プラン間の機能差の詳細
- 管理者向け機能

> 参照した公開情報: 公式サイト(herazika.com、検索結果経由)、note のレビュー記事、
> 東京新聞の報道記事、資格学校TACの提携紹介ページ、個人ブログのレビュー。
> 名称・文言・デザイン・キャラクター等は一切流用していない。

## 2. 本サービス「MokuTomo(モクトモ)」の機能一覧

コンセプト: **「黙々、でもひとりじゃない。」** — やる気に頼らず、ボタン1つで
"誰かと一緒に勉強している状態" に入れるオンライン自習室。

### MVP(Phase 2 実装済み範囲)

| カテゴリ | 機能 |
|---------|------|
| 認証 | メール+パスワード登録 / ログイン / ログアウト / パスワード再設定 / 規約同意 / 退会(アカウント削除) / 利用停止ユーザーの入室拒否 |
| オンボーディング | サービス説明 → カメラ/ぼかし/マイク不使用の説明 → 権限確認 → カメラ確認 → 表示名 → 今日の学習内容 → 体験セッション(5分) |
| ホーム | 今すぐ入室 / 次回予約 / 本日・今週の集中時間 / 今週のコマ数 / 連続日数 / 今日の目標 / 現在自習中の人数 / 最近の履歴 / お知らせ / カメラテスト導線 |
| 入室前 | 学習内容入力 / 集中時間選択(5/15/25/50分、標準25分) / カメラプレビュー(ぼかし確認) / カメラ選択 / ルール同意 |
| 自習室 | WebRTCメッシュによる映像共有(最大6名) / 送信映像自体への強制ぼかし(Canvas加工) / マイク完全無効 / グリッド自動調整 / 表示名・学習内容・接続状態表示 / カメラOFFプレースホルダー / 空き部屋自動割当・満室時新規作成 / 入退室リアルタイム反映 / リロード後の再接続 / 多重入室防止 / 終了済みセッションへの再入室防止 |
| タイマー | サーバー時刻基準の同期タイマー / 開始前カウントダウン / 残り時間表示 / 終了画面 / ブラウザ通知 / 通知音(任意) / 一時停止不可 / 途中退出は確認ダイアログ付き / リロード後の残り時間復元 / 案内役「ともしび」の開始・終了メッセージ |
| セッション終了 | 完了メッセージ / 集中時間 / 学習内容 / 本日・今週累計 / コマ数 / 自己評価(3択) / 振り返りメモ / もう1コマ / 休憩 / ホームへ |
| 記録 | 正常終了のみ完了コマとして記録 / 途中退出は実参加時間を別途記録 |
| 履歴 | 日別/週別/月別集中時間 / 完了コマ数 / 途中退出数 / 連続日数 / 曜日別・時間帯別傾向 / 学習内容履歴 / カレンダー / 棒グラフ / 目標達成率 / 空状態デザイン |
| 予約 | 単発予約CRUD / 毎週繰り返し予約 / 開始前ブラウザ通知 / 予約時刻のホーム強調表示 / 予約から直接入室 / 欠席記録 / タイムゾーン対応 / 無料プランは3件まで |
| 習慣化 | 学習時間に応じたXP(サーバー側で付与) / レベル / 連続日数 / 週間・月間目標 / バッジ(初回・10コマ・100コマ・7日・30日) / 達成演出 |
| 応援者 | メール招待 / 応援者の同意 / 開始・完了通知 / 週間レポート / 共有停止 / 削除 |
| 安全 | 通報(5分類) / ブロック(同室回避) / レート制限 / CSRF・XSS・SQLi対策 / RLSによる認可 / 管理者権限チェック |
| 管理者 | 管理者ログイン / ユーザー一覧・検索・利用停止・解除・強制退室 / 部屋一覧・参加人数 / 通報対応 / お知らせCRUD / 利用状況集計(日別利用者・MAU・総集中時間・完了数) / 管理操作ログ |

### 正式版(Phase 3 以降)

- LINE Messaging API連携(通知・ダイジェスト・レポート)
- メール通知の完全自動化(予約リマインダーのcron送信)
- 詳細な学習分析(集中度スコア、比較分析)
- Googleログイン(Supabase AuthのProvider追加のみで対応可能な設計)
- SFU(LiveKit)への移行による大人数ルーム
- プロフィール画像アップロード(Supabase Storage)
- Sentryによる本番エラー監視の本組み込み
